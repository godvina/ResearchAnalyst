How It's Used

Known crime typology indicators from past prosecutions are embedded and indexed. New case evidence is embedded with the same model, then k-NN surfaces which historical patterns the evidence most resembles.

Example: New financial docs → Titan Embed → k-NN returns 0.91 similarity to "Human Trafficking → Financial Control" → flag for investigator
What is 0.91? Cosine similarity score (0–1). 1.0 = identical meaning, 0.0 = completely unrelated. 0.91 = very high confidence this evidence matches a known prosecution pattern.
